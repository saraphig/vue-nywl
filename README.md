# vue-class

> A Vue.js project

### 版本要求
- node >= 7.0.0
- brower > ie8

### 开发规范
- Airbnb

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
